
## 📝 Usage

### Single Post Analysis
1. Edit any post or page
2. Find "AI Content Analyzer" meta box
3. Select analysis options
4. Click "Analyze Content"
5. Review and apply results

### Bulk Processing
1. Go to Tools > Content Analysis
2. Configure content selection
3. Choose analysis features
4. Set processing options
5. Start bulk analysis

### Entity Management
1. Go to Tools > Named Entities
2. View entity statistics
3. Filter and search entities
4. Manage entity categories
5. Export/import data

## 🛠️ Technical Details

### Database Tables
- Uses WordPress taxonomy system
- Custom meta fields for entity data
- Queue table for bulk processing

### API Integration
- DeepSeek API for AI analysis
- Fallback to basic analysis
- Rate limiting and error handling

### Performance
- Batch processing system
- Configurable processing speeds
- Memory optimization
